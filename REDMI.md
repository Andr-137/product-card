# Тестовое задание
## Функциональность
### Проект реализует интерфейс фильтрации карточек продуктов с помощью вкладок:
1. Фильтрация по категориям:
- Marketing
- Management
- HR & Recruting
- Design
- Development
- All (все категории)
2. Поиск по карточкам - фильтрация по тексту
3. Интерактивные элементы:
- Подсчет количества карточек в каждой категории
- Плавные анимации при переключении
- Эффекты наведения
## Что реализовано
1. Модульная структура SCSS
- Четкое разделение на компоненты
- Использование миксинов и функций для повторного - использования кода
- Поддержка адаптивности через кастомные функции
2. Адаптивный дизайн
- Использование clamp() для плавного изменения размеров
- Медиа-запросы для разных брейкпоинтов
- Флексбокс для сетки карточек
3. JavaScript логика
- Фильтрация карточек по категориям
- Динамический подсчет количества
- Поиск по тексту
- Плавные анимации
4. UI/UX элементы
- Визуальная обратная связь при наведении
- Активное состояние вкладок
- Четкая иерархия информации в карточках
## Недостатки и проблемы
1. Добавить ARIA-роли для табов, aria-hidden для декоративных иконок, alt-тексты
2. БЭМ может быть намного лучше, последнее время пользовался TailwindCSS.
3. Добавить loading="lazy" к изображениям, использовать content-visibility для карточек внизу, вообщем подключены, что бы были.
